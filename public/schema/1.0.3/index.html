Schema version 1.0.3

Schema Files (Turtle format)

  • Brick TagSets: Entity classes with a hierarchy. Major component of Brick.
    • E.g., Temperature_Sensor is a subclass of Sensor
  • Brick Relationships: Brick relationships and meta-structure constructing TagSets with Tags.
    • E.g., sensor1 is a point of vav1.
  • Brick Tags: Tags are subcomponents of all TagSets.
    • E.g., Temperature_Sensor consists of Tempearture and Sensor.
  • Brick Dimensions: Information dimensions of TagSets.
