Brick Structure

Brick adheres to the RDF data model (Resource Description Framework), which represents knowledge as a graph expressed as tuples of (subject, predicate, object) known as triples.

The Brick examples and most of our development has used the Python rdflib library.

Applications and services can query the RDF representation of a building using SPARQL (SPARQL Protocol and RDF Query Language).

More detail and some examples will follow soon.
