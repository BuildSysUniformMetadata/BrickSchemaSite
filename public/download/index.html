Brick Downloads

Brick consists of several TTL (“Turtle”) files. Latest version (v1.0.2) available here:

  • Brick.ttl: Brick classes and tagsets
  • BrickFrame.ttl: Brick relationships
  • BrickTag.ttl: Brick tags (internal)
  • BrickUse.ttl: Brick uses relationships (internal)

Past releases are available on GitHub
